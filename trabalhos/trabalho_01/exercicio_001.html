1 - O que é HTML?

HTML (HyperText Markup Language), é uma linguagem de marcação usada, essencialmente, na estrutura do conteúdo de um website.
Para além disso, HTML é muitas vezes confundida como linguagem de programação, no entanto, HTML, é uma linguagem de marcação.

2 - O que é um elemento HTML? Dê exemplos.

O elemento é a combinação entre a tag de abertura, conteúdo e, se for aplicável, a tag de fecho.
Este é um exemplo de elemento de parágrafo <p>Eu sou um parágrafo</p>.
Este é um exemplo de elemento com tag vazia <img src="bob.jpg" alt="bob">.

3 - Que ferramentas são necessárias para construir um site HTML?

Para construir um site HTML são necessárias ferramentas como o editor de texto, navegador web e conexão à internet. O editor de texto (ex:VSCode), é fundamental para escrever e editar o código. Navegadores de web como google chrome, opera, etc, são importantes para podermos visualizar e testar o website. Por fim, a conexão à internet é fundamental para ser possivel o acesso ao site que estamos a criar.

4 - Quais os elementos base necessários para a criação de uma página HTML?


    <!DOCTYPE html>
    <html lang="pt">

        <head>
            <meta charset="UTF-8">

            <meta name="viewport" content="width=device-width, inicial-scale=1.0">

5 - O elemento "br" necessita ser fechado?

Não, o elemento <br> é uma tag vazia, ou seja, como não tem conteudo interno não precisa de ser fechada.

6 - Para que servem os atributos? Dê um exemplo.

Os atributos em HTML são usados para darem mais informação a um elemento e, para além disso, ajudam a controlar o comportamento e aparência do mesmo.
Exemplo: "type="A"" (atributo usado na criação de listas ordenadas por letras maiúsculas do alfabeto)
